Shibedia Usage Guide

Welcome to Shibedia, the platform to create, manage, and explore Meme Coins on Shibarium! This guide explains how to launch a Meme Coin, add metadata, and browse tokens.

Getting Started

Shibedia lets you create and view Meme Coins on Shibarium. No wallet is needed to browse, but you’ll need one to create or manage tokens.

What You Need

  • Web Browser: Chrome, Firefox, or any modern browser.
  • MetaMask: Install MetaMask for creating or managing Meme Coins.
  • BONE: Shibarium’s currency for transaction fees.

How to Use Shibedia

1. Create a Meme Coin

Launch your own Meme Coin on Shibarium.

  1. Visit https://shibfun.github.io/Token/.
  2. Click Connect Wallet to link MetaMask to Shibarium.
  3. Enter a name (e.g., “Paw”) and symbol (e.g., “PAW”).
  4. Click Create Meme Coin, confirm in MetaMask (requires BONE), and note the contract address. You’ll receive 100 trillion tokens.

2. Add Token Metadata

Enhance your Meme Coin with an image, description, and links.

  1. Go to https://shibfun.github.io/Token/SetInfo/.
  2. Connect MetaMask (must be the token’s owner).
  3. Enter your Meme Coin’s contract address and click Verify Owner.
  4. Add:
    • Image URL: Square image (e.g., https://example.com/token.png).
    • Description: Short text (e.g., “Paw is a meme coin on Shibarium.”).
    • Website: Your site (e.g., https://example.com).
    • Twitter: Twitter link (e.g., https://x.com/pawcoin).
    • Telegram: Group link (e.g., https://t.me/pawcoin).
  5. Click Set Metadata, confirm in MetaMask, and view your token on Shibedia.

3. Browse Meme Coins

Explore Meme Coins on Shibedia.

  1. Open https://shibfun.github.io/Token/list/.
  2. Browse:
    • View up to 100 recent Meme Coins, each with an image, name, description, and links.
    • No wallet needed.
  3. Search:
    • Enter a Meme Coin’s contract address in the search bar.
    • Click Search to view details.
    • Invalid address? You’ll see “Invalid contract address!”

Troubleshooting

  • Can’t Connect Wallet: Install or update MetaMask, add Shibarium (chain ID: 109), and retry.
  • Creation Fails:
    • Ensure sufficient BONE for gas fees.
    • Check network status at ShibariumScan.
  • No Tokens Show: Refresh the page or check your internet connection.
  • Search Fails: Verify the contract address is correct.
  • Metadata Not Updating: Ensure you’re the token owner and confirm the transaction in MetaMask.
